# Accuracy Improvements — Medium Impact

Deze stories brengen je van ~95% naar ~97-98% compatibiliteit. Specifieke games hebben deze features nodig.

## Volgorde

| # | Story | Impact |
|---|-------|--------|
| 1 | [16.1 — RTC (Real-Time Clock)](16.1-rtc.md) | Pokémon dag/nacht, berry groei |
| 2 | [16.2 — Open Bus Behavior](16.2-open-bus.md) | Anti-piracy checks, edge cases |
| 3 | [16.3 — Advanced Sprite Rendering](16.3-advanced-sprites.md) | Semi-transparent OBJ, pixel limit, forced blank |
| 4 | [16.4 — BIOS Read Protection](16.4-bios-protection.md) | BIOS memory read-after-SWI behavior |

## Wanneer beginnen?

Als specifieke games die je wilt draaien niet correct werken. RTC is de eerste prioriteit als je Pokémon games wilt spelen.
